You should notice:

  1. This iframe covers the entire viewport
  2. The document in the parent window cannot be scrolled while this iframe is open
  3. The "Close" button below auto-focuses
  4. If you , the iframe will reload and the "Close" button will be re-focused
  5. If you press tab (or shift+tab), focus will wrap between the inputs in this iframe
  6. You can click the "Push nested child" button to push another layer.
  7. After closing this window, the button you clicked to open it will be re-focused
